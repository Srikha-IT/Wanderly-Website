# Wanderly - Travel Website

Wanderly is a responsive travel website built using HTML, CSS, and JavaScript. It provides users with an attractive interface to explore destinations, search for trips, and learn more about travel services.

## Features

- Responsive design for desktop and mobile devices
- Home page with automatic background image slider
- About page with company information
- Explore page displaying travel destinations
- Contact form with JavaScript validation
- Login page with email validation
- Search functionality using Local Storage
- Interactive user interface with JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
